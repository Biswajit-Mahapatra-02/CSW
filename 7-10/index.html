HTML

It stands for Hyper Text Markup Language. It is the structural aspect of the website. It provides the content of it.

CSS

It stands for cascading style sheets. It is the behavioural part of the website. It helps design/beautify the html content.

JS

It stands for JavaScript. Unlike html and css, its an actual programming langauge. It helps make website more lively. It helps make the website dynamic and change according to the sitations or the client's needs. Some examples:

  • Validation operation:
    • For example, while filling a from if client enters some unexpected values which is not acceptable by the server and may cause fatal problems. JS can help prevent that by not allowing the client to submit the form with the wrong values.
    • Account password validation.
  • It can also take command of the backend programming using the node.js architecture

jQuery

A JS library that helps making website dynamic. It has some predefined functions that helps making websites dynamic easy.

Why use jQuery?

  • It helps writing lesser more efficient code than JS. Write less do more.
  • It is cross platform
