Hyper Text Markup Language or HTML

HTML is the content of the web page. The barebones.

The format of HTML is follows:

< tag > Content < / tag >
Different types of HTML tags
tag tag name description working
< a > anchor tag Used to create a hyperlink in the site An anchor tag
< strong > strong tag Used to make the letters inside the tag bold A strong tag usage
< em > emphasis tag Used to make the letters inside the tag italic A em tag usage
< p > paragraph tag Used to distinguish a paragraph from a document

A p tag usage
< quote > quote tag Used to distinguish a quote from a document A quote tag usage
