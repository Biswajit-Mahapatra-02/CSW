Cascading Style Sheets or CSS

This takes care of the looks of the website. The beautification of it.

There are 3 types of css selectors

  • Element selector: This selector selects the html element directly by its name. The general html tag.
  • Class selector:This selector selects the html element by its specified class. This is the best way to select any html element. To mention a class selector in a css file then start the name with a period. "."
  • ID selector: This slector selects the html element by its specified id. To mention an id in a css file, start the id name with hash symbol. "#"
